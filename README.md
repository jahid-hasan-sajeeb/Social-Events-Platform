<h1> SocialServe </h1>

A simple social development event platform where users can browse upcoming events, create events, manage events, join events, and update their profile.

<h1>Live Links</h1> 


Live Website: https://social-event-jahid.netlify.app/

Client Repo: https://github.com/jahid-hasan-sajeeb/Social-Events-Platform

Server Repo: https://github.com/jahid-hasan-sajeeb/Social-Event-Backend

Live Server: https://a-10-back.vercel.app/

<h1>Features</h1>

User Registration, Login, Logout

Google Login

Reset password via email

Create, update, and manage events

Join events

Upcoming events page with filters

Profile page (update name & photo)

Protected routes

Dark theme UI

Smooth animations

<h1>Tech Used</h1>

Frontend:
React, React Router, Firebase Auth, Tailwind CSS, React Toastify, Framer Motion

Backend:
Node.js, Express, MongoDB
